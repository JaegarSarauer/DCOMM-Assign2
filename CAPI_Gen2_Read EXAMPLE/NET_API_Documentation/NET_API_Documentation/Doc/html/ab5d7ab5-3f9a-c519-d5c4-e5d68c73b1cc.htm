loop (Boolean)
True indicates the reader should inventory tags until explicitly told to stop
itd (InventoryTagDelegate)
Delegate to call when a tag is inventoried. Any long running or computationally intensive work should be done outside of the this function call so as not to impede the inventory process.
context (Object)
Context object passed through to delegate call
Return Value
True if the call to InventoryTags was terminated by a call to the InventoryTagDelegate, false if terminated because of another reason
